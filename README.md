# Appen :fountain_pen: :penguin:

Eccomerce de canetas

## Rodar localmente

Clone o repositório

```bash
  git clone https://github.com/MarcioAjunior/trabalho_containers.git
```

Vá para o diretório do app next

```bash
  cd pen
```

Instale as dependencias

```bash
npm install
```

Inicie a aplicação

```bash
npm run dev
```
